# Kamus Mandarin Pribadi

Website static untuk menyimpan dan belajar kosa kata Mandarin.

## Cara pakai
1. Clone atau buat repo baru di GitHub.
2. Tambahkan file `index.html`, `styles.css`, dan `vocab.json` (isi contoh ada di repo).
3. Aktifkan GitHub Pages (Settings → Pages → source: `main` branch / root) — lalu buka `https://<username>.github.io/<repo>/`.
4. Untuk menyimpan perubahan permanen:
   - Edit `vocab.json` lewat GitHub web UI (Recommended, aman), atau
   - Gunakan fitur **Simpan ke GitHub** di halaman (memerlukan Personal Access Token — jangan dipublikasikan).
5. Untuk menyimpan lokal di browser: klik **Simpan ke Local** atau gunakan `Ctrl/Cmd+S`.

## Keamanan
- Jangan simpan Personal Access Token di repo publik.
- Menyimpan ke GitHub lewat token dari browser berarti token ada di memori browser — gunakan hanya jika kamu paham risikonya.
